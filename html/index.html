Instructions

This site provides a calendar feed which can be consumed by calendar programs. In other words, you won't see the movie listing anywhere on this site, you have to tell Google Calendar (for example) to subscribe to it. The feed is available at https://seattle-movies.innocence.com/film_calendar.ics. If you'd rather add one calendar per theater, you can find per-theater calendars under "Per-Theater Feeds" in the navigation bar at the top of the page.

About This Project

I'm horrible at keeping track of movies I want to see. I subscribe to all the local cinema newsletters but that's not enough for me. Thus, I wrote some code to scrape various cinema web sites and generate a calendar. It seemed as though this might be useful to other people so now there's a Web site.

Please consider buying memberships to cinemas that sell them -- many of these are non-profits which deserve our support!

And always remember, Seattle has one of the remaining great video stores in the world, Scarecrow Video.
